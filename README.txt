Готовый комплект для недели истории

Что внутри:
- index.html — готовая страница сайта с 10 разделами по истории космоса.
- папка assets — изображения для разделов.
- папка qr — QR-коды для каждого раздела.
- qr_manifest.json — список QR-кодов и ссылок.

Важно:
Сейчас QR-коды созданы с ШАБЛОННЫМ адресом:
https://ВАШ-АДРЕС-САЙТА/space_week/index.html#...

Чтобы QR-коды заработали по-настоящему, нужно:
1. Загрузить папку сайта на хостинг, GitHub Pages или другой сервис.
2. Узнать точный адрес страницы index.html.
3. Либо пересоздать QR-коды под свой адрес.

Если хочешь быстро пересоздать QR-коды, замени базовый адрес в qr_manifest.json или в любом генераторе QR-кодов.

Пример рабочего адреса:
https://my-school-project.github.io/space_week/index.html#copernicus

После этого можно распечатать QR-коды и развесить по школе.
